

Overview
The Job Board Application facilitates the hiring process with features for job postings, candidate applications, and profile management.

Key Features
For Candidates (Job Seekers):
Search jobs with filters (location, category, salary).
Manage profiles with resume and document uploads.
Notifications for job updates and application statuses.
For Recruiters (Employers):
Post, edit, and manage job listings.
Search for candidates by skills, location, and experience.
Review applications and communicate with candidates.
For Administrators:
Manage users and job postings.

Technologies Used
Backend:Python and  Django
Frontend: HTML, CSS,js, Bootstrap
